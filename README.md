
# 📊 Scientific Programming Web App
**_Group Members : Mehwish Zehra_** **|**  **_Arfeen_** 
**Live :** **https://numeralab.streamlit.app/** 
<br>

An interactive **Scientific Programming Web Application** built using Python and Streamlit to solve and visualize core numerical methods.

This project allows users to perform complex numerical computations through an intuitive UI and understand results via graphical visualization.

---

## 🚀 Features

* 🧮 Solve numerical problems interactively
* 📊 Visualize results with graphs
* 🎯 User-friendly interface (no coding required)
* ⚡ Fast computation using optimized Python logic
* 🧩 Modular design (4 labs, 3 methods each)

---

## 🧪 Implemented Labs & Methods

### 🟢 Lab 1: Root Finding

* Bisection Method
* Newton-Raphson Method
* Secant Method

👉 Solve equations of the form **f(x) = 0**

---

### 🟡 Lab 2: Interpolation

* Lagrange Interpolation
* Newton Forward Difference
* Divided Difference Method

👉 Estimate unknown values between data points

---

### 🟢 Lab 3: Numerical Integration

* Trapezoidal Rule
* Simpson’s Rule
* Midpoint Rule

👉 Approximate area under curves

---

### 🟡 Lab 4: Differential Equations

* Euler Method
* Modified Euler Method
* Heun’s Method

👉 Solve first-order differential equations numerically

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/scientific-programming-app.git

# Navigate to project folder
cd scientific-programming-app

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

## 📈 How It Works

1. User selects a **lab**
2. Chooses a **method**
3. Enters required inputs (function, values, etc.)
4. System computes results using numerical algorithms
5. Outputs:

   * ✅ Final result
   * 🔁 Iterations (if applicable)
   * 📊 Graph visualization
